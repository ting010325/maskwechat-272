这是一个微信密友模块
📲 使用方法

① 安装并激活模块

安装本模块

在 LSPosed / Xposed​ 中勾选 微信


重启手机或重启微信



我们的频道: https://t.me/MaskWechat3

✨ 功能特性

🔒 隐藏指定联系人
​
将指定好友从会话列表、通讯录中隐藏，保护隐私。

🚫 屏蔽朋友圈

隐藏指定联系人的朋友圈入口及动态。


📞 屏蔽语音通话
​
拦截指定联系人的语音通话请求。

📹 屏蔽视频通话
​
拦截指定联系人的视频通话请求。

👥 隐藏通讯录
​
指定联系人不在通讯录列表中显示。

🔍 隐藏主页搜索
​
禁止通过微信首页搜索找到指定联系人。

💬 隐藏聊天记录搜索​

在全局聊天记录搜索中屏蔽指定联系人消息。

📲 模块入口

打开 微信 → 我 → 设置 → 更多 → 关怀模式​

👉 长按「开启」​ 进入密友管理界面

💬 使用方式

在 聊天页面输入命令：

#add

即可将当前联系人加入密友列表

长按好友​ 直接弹出「添加密友」菜单
